Download all the packages by typing in the console "npm install", make sure you have node before hand.
Type in the console "node server" to start the app and navigate to http://localhost:3000/ on your browser to view the app.
The messages aren't stored in any type of database so each time you refresh the page the previous messages will disappear.
Open the same page on a different tab or windows to check that you can send a message to other people on the same page.
You'll also see that a line in the chat says the name of anyone who is currently typing out a message.
